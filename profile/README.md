# 🚀 AstroMerchant

Build lightning-fast, modern eCommerce storefronts with Astro.

AstroMerchant is an open-source eCommerce framework designed for developers who want full control, outstanding performance, and seamless integrations.

## ✨ Features

- ⚡ Astro-powered storefronts
- 🛒 Stripe integration
- 📦 File-based product management
- 🎨 Tailwind CSS
- 🌙 Dark mode
- 📈 SEO optimized
- 🔌 Extensible architecture

## 🚀 Quick Start

```bash
git clone https://github.com/AstroMerchant/astromerchant.git

cd astromerchant

npm install

npm run dev
```
