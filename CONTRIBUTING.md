# Contributing to AstroMerchant

Thank you for your interest in contributing to AstroMerchant.

## Getting Started

1. Fork the repository.
2. Create a feature branch.

```bash
git checkout -b feature/my-feature
```

3. Make your changes.
4. Write or update tests where appropriate.
5. Commit using clear commit messages.
6. Push your branch.
7. Open a Pull Request.

## Reporting Bugs

Please use the Bug Report template and include:

* Description
* Steps to reproduce
* Expected behavior
* Screenshots (if applicable)
* Environment details

## Feature Requests

When suggesting a feature, please explain:

* The problem you're solving
* Your proposed solution
* Alternative approaches considered
* Additional context

## Development Guidelines

* Keep pull requests focused.
* Follow existing coding conventions.
* Update documentation when necessary.
* Ensure CI checks pass before requesting review.

We appreciate every contribution, no matter how small.
