# AstroMerchant Community Health Files

This repository contains the default community health files for the AstroMerchant GitHub organization.

These files are automatically used by repositories that do not define their own community health files.

## Included

- Code of Conduct
- Contributing Guide
- Security Policy
- Support Guidelines
- Issue Templates
- Pull Request Template
- GitHub Actions Workflows

Learn more:
https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file
