# Security Policy

## Supported Versions

Security updates are provided for actively maintained releases of AstroMerchant.

## Reporting a Vulnerability

Please do not disclose security vulnerabilities publicly.

Instead:

1. Contact the maintainers privately.
2. Include detailed reproduction steps.
3. Provide any proof-of-concept if available.
4. Allow time for investigation before public disclosure.

We appreciate responsible disclosure and will acknowledge reports as quickly as possible.
